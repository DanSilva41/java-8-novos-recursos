Default methods:
    Métodos que contém um corpo(implementação) default dentro de uma interface. EX: default void sort() na interface java.util.List
Interface funcional:
    Onde tenho somente um método não implementado e posso ter outros default methods.
    É possível usar lambda por cima da interface pois se referenciará este únido método contido na interface.
Method reference:
    É a forma de escrever uma expressão llambda mas de forma mais enxuta. Ex: String::lenght
Stream:
    método adicionada às coleções. Filtrar, definir criterios de busca
Optinal:
    Nova classe como alternativa para evitar NullPointerException
Datas:
    Os principais métodos do Java 8 não modificam a variavel atual, mas sim retornam uma nova.
